sublime-js-console-snippets
===========================

I was fed-up writing the whole console.blablabla and wrote couple of simple commonly used snippets.


## Methods Covered
- console.assert()
- console.count()
- console.debug()
- console.error()
- console.group()
- console.groupEnd()
- console.info()
- console.log()
- console.time()
- console.timeEnd()
- console.warn()
